# Smart Complaint App

## Overview

Smart Complaint App is an NLP-based complaint management system that automatically classifies complaints and predicts their urgency level.

The system helps organizations analyze complaints efficiently and prioritize critical issues.

## Features

- Complaint Classification
- Sentiment Analysis
- Urgency Detection
- User Registration & Login
- Complaint Tracking
- Complaint History Management

## Technologies Used

- Python
- Streamlit
- NLTK
- TextBlob
- Scikit-Learn
- Pandas

## Project Structure

- app.py → Main application
- train.py → Model training script
- complaints.csv → Training dataset
- model.pkl → Trained model
- vectorizer.pkl → Text vectorizer

## How to Run

1. Install requirements

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
streamlit run app.py
```
